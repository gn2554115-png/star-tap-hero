Privacy Policy

Star Tap Hero is a simple reflex game built for the Pi Browser.

  • We do not collect or store personal data on our own servers.
  • Any Pi account information is handled via the official Pi SDK inside Pi Browser.
  • Analytics, if enabled in the future, will only be used to improve gameplay.
  • You can stop using the game at any time; no additional data is kept by us.

If you have any questions, please contact us via the email listed in the Pi Developer Portal.
